# isc-toilet-search

学内のトイレ利用の利便性を高めるために作成した、トイレ利用感知システムです。
Obnizと各種センサーを使用してリアルタイムの空き状況を取得し、ブラウザから確認できます。

## 🛠 技術スタック
- **Backend**: PHP, MySQL
- **Frontend**: HTML, CSS, JavaScript (jQuery)
- **Hardware**: Obniz Board, 距離センサー (GP2Y0A21YK0F), 磁気センサー

## 📂 ファイル構成
| ファイル名 | 役割 |
| :--- | :--- |
| `admin.html` | Obniz・センサーをサーバーと接続・制御するための管理ページ。 |
| `toire.js` | センサーからデータを取得し、サーバーへ送信するメインロジック。 |
| `save_data.php` | 受信データから「使用中」を判定し、DBへ保存するプログラム。 |
| `get_data.php` | DBの最新情報を取得し、JSON形式でクライアントへ返す。 |
| `db_config.php` | サーバーの環境変数からDB接続情報を取得する。 |
| `index.html` | 一般ユーザー用の表示確認ページ（Obniz接続処理を分離）。 |

## 💡 こだわりポイント
- **ユーザー体験の最適化**: 
  Obnizの仕様によるコンソールログの警告がとてもうるさいので、管理者用(`admin.html`)と閲覧用(`index.html`)を物理的に分離しました。
- **データ取得の効率化**:
  `get_data.php` では、SQLの `GROUP BY` を使用して、各センサーの最新状態のみを効率的に取得するように設計しています。
